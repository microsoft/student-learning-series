# Trail Journal media kit (prepared pre-recording)

This folder is the prepared local media source for Episode 06.

## Scope
- Exactly 8 Pacific Northwest day-hike images.
- Parks used: Olympic, Mount Rainier, and North Cascades National Parks.
- Preference applied: landscape-first selection without identifiable visitors.

## Rights and sourcing
- Each image is tied to its own NPS source page (`/media/photo/view.htm?id=...`).
- `sources.csv` records both the source page URL and direct image URL.
- Public-domain basis is verified per image from NPS asset metadata:
  `ConstraintsInformation.Constraint = Public domain` and `GrantingRights = Full`.
- The credited creator/byline from each source asset is captured in `sources.csv`.
- Third-party or unclear-rights images are intentionally excluded.

## Folder structure
- `photos/<location>/...` - 8 selected images grouped by park/location.
- `sources.csv` - location, local file path, source page URL, image URL, creator credit, public-domain basis, caption, and alt text.

## Episode 06 usage
- Use scoped local-file search against this folder only.
- Select images and captions from this pack when creating the Trail Journal project page.
