# Student AI Learning Series PR Quality Gates

Apply these gates to every episode PR (01-10) before merge.

## Required gates (all PRs)
- [ ] 5–7-minute demo flow is demonstrable from files in the episode folder.
- [ ] Visible delta from previous episode output is clear.
- [ ] Source traceability is documented for factual/media content.
- [ ] Simulated-data labeling is explicit wherever simulated data is used.
- [ ] GitHub Pages base-path behavior is correct for local/hosted routes.
- [ ] Accessibility and mobile behavior are reviewed.
- [ ] Fallback behavior is documented.
- [ ] Handoff files and routes are explicitly listed.

## Additional gate emphasis by episode
- Episode 05: PDF embed fallback and download link validated.
- Episode 06: Media rights basis and per-image source metadata validated.
- Episode 08: Deck output traceable to approved Campus source pack.
- Episode 09: Bootstrap-only library constraint confirmed.
- Episode 10: PR review workflow evidence captured for cloud-agent handoff.
