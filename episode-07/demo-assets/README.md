# Episode 07 demo assets

Filming artifacts for Episode 07 of the Student AI Learning Series.

See [`word-to-blog-workflow.md`](./word-to-blog-workflow.md) for the on-screen contract that
governs the manual demo flow, prompt drafting, and `publish-blog` skill packaging. That file is
the authority if any other document conflicts.

## Folder layout

| Folder | Contents |
| --- | --- |
| `word-notes/` | Approved Word source documents (inputs; not modified on camera) |
| `markdown-drafts/` | Markdown drafts produced from Word notes; reviewed before HTML generation |
| `critique/` | Rubber Duck critique output captured per post |
| `preview-html/` | Generated HTML post files previewed before publish approval |
| `publish-blog/` | The `publish-blog` skill file created after the manual flow |
