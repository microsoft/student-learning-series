# Jamie Rivera portfolio

This is the frozen Episode 7 starting website. It uses plain HTML, CSS, and JavaScript with no build step.

## Blog folder contract

- Save authored Markdown drafts in `blog/drafts/<post-slug>.md`.
- Generate approved static post pages in `blog/posts/<post-slug>.html`.
- Add a published post link to `blog/index.html` only after the draft is approved.
- Keep links relative so the website works from a local HTTP server and from a repository subpath.

Drafts are source material and are not linked from the website. The starting state intentionally contains no published posts.
